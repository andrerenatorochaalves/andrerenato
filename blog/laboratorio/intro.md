# Ensaios de Laboratório

Nesta seção, abordo os principais ensaios de laboratório utilizados em geotecnia de mineração. 

---

## Importância

Os ensaios de laboratório são fundamentais para: 

1.  Caracterizar os materiais geotécnicos
2. Obter parâmetros para análises numéricas
3. Avaliar comportamento sob diferentes condições de carregamento
4. Verificar suscetibilidade à liquefação
